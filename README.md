# Student Score Dashboard

A simple JavaScript dashboard project built for Week 1 of the FlexiSAF Frontend Internship.

This project demonstrates the use of core JavaScript concepts including:

- ES6 syntax
- Arrow functions
- Arrays and objects
- Array methods (`map`, `filter`, `reduce`, `forEach`)
- DOM manipulation
- Variable scope using `let` and `const`

## Features

- Display student information dynamically
- Calculate average student score
- Filter passed students
- Render student cards using JavaScript
- Simple responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## JavaScript Concepts Implemented

### Array Methods

- `map()` → Used to render student cards
- `filter()` → Used to filter passed students
- `reduce()` → Used to calculate average score
- `forEach()` → Used for logging student data

### ES6 Features

- Arrow functions
- Template literals
- `let` and `const`
- Object and array usage
